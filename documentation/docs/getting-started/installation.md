Make sure you have `yarn` or `npm` installed before proceeding.
<br/><br/>

##### NPM:

```bash
npm install -g react-devcli
```

<br/>

##### YARN:

```bash
yarn global add react-devcli
```

Make sure the package is installed successfully by running

`react --help`
